Snowride 1.2.1

  • Find Usages (for user keywords only).
  • Snowride now offers documentation and go-to-definition for keywords in external folders.
  • Fixed a bug that caused autocompletion to not show keywords in some cases.
  • Offers you to reload files and folder when they're changed from outside Snowride.
  • Fixed some parser errors (BOM, ellipses)./li>
  • Keywords are now recognized and highlighted within FOR loops.
  • You now have the option of disabling parsing .txt files.
  • Fixed some bugs for template test cases.
  • Fixed some bugs during reloading
  • Documentation popup is positioned properly on the screen even if it's near the edge of a screen.

Snowride 1.1

  • Adds additional folders from Settings to --pythonpath.
  • Fixed a bug that put extra newlines into files.
  • Added "Comment Out" and "Uncomment" keyboard shortcuts.
  • Fixed some parser grammar issues.
  • Prettier documentation, and with larger font size.
  • Keywords are ordered by kind.
  • Arguments show up highlighted based on whether they're mandatory, optional and/or filled in correctly/incorrectly.
  • You can click through to imported resources.
  • Resources and libraries that Snowride can't find are highlighted in red.
  • Added the button "Reload all".

Snowride 1.0

  • Fixed a couple of bugs that triggered dialog boxes in some scenarios.
  • Added the changelog and some help options.
  • Changed the font in my boxes to a "system default" so it always shows up as a monospace font.
  • Performance improved.
  • Snowride now supports .txt files, but it can only create .robot files.
  • Text improvements across the interface. More understandable error messages.
  • Snowride now understands Python keywords and keywords imported from XML.
  • Fixed a bug that caused extra whitespace to appear in files.
  • Fixed a bug that caused go-to-definition to only focus the target in the treeview, but not switch to it.
  • Fixed a bug that caused "navigate back" functionality to be polluted with tests you never entered
  • Added list of arguments to quick documentation.
